    Direccion

    Representa una dirección de un estudiante.

    Constructor

    new Direccion(calle, numero, piso, codigoPostal, provincia, localidad)

    Crea una dirección con los datos proporcionados.

    • Se valida el código postal (5 números).
    • Si no es válido, se establecerá como "00000".
